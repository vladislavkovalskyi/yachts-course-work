### **Frontend**

* Install all dependencies: `npm i`
* Run project in development mode: `npm run dev`
* Run project in stable (production) node: `npm run start`

### **Backend**

__Hint: Do not forget to check your `backend/includes/Database.php` file__

* Install **PHP**
* Run Database Service (**MySQL**)
* Run PHP Server: php -S localhost:8080
